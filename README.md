# Kirana Insights

A decision support engine for retail inventory management.

## Features
- Optimal Reorder Quantities
- Demand Forecasting (ARIMA)
- Market Trend Monitoring (Mandi Prices)

## Setup

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn app.main:app --port 8002`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
